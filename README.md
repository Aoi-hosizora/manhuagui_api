# manhuagui-backend

+ An unofficial backend for manhuagui (https://www.manhuagui.com/) written in golang/gin

### Endpoints

+ `GET /v1/manga`
+ `GET /v1/manga/:mid`
+ `GET /v1/manga/:mid/:cid`
+ `GET /v1/list/serial`
+ `GET /v1/list/finish`
+ `GET /v1/list/latest`
+ `GET /v1/list/updated`
+ `GET /v1/category/genre`
+ `GET /v1/category/zone`
+ `GET /v1/category/age`
+ `GET /v1/category/genre/:genre`
+ `GET /v1/search/:keyword`
+ `GET /v1/author`
+ `GET /v1/author/:aid`
+ `GET /v1/author/:aid/manga`
+ `GET /v1/rank/day`
+ `GET /v1/rank/week`
+ `GET /v1/rank/month`
+ `GET /v1/rank/total`
+ `GET /v1/comment/manga/:mid`
